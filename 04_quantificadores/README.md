# Quantificadores

| Quantificador | Significado           |
| :-:           | -                     |
| {2,3}         | De 2 a 3              |
| {3,}          | 3 ou mais             |
| {2}           | Exatamente 2          |
| {0,1}         | 0 ou 1 ~ semelhante ? |
| {0,}          | 0 ou + semelhante a * |
| {1,}          | 1 ou + semelhante a + |
